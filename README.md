# Princeton University's Algorithms Part 1 Course 
A collection of data structures and algorithms in Java.
Coursework from the [Algorithms Part 1 Course](https://www.coursera.org/learn/algorithms-part1) covering topics such as:
- Union-Find
- Analysis of Algorithms (Ex: Space & Time Complexities)
- Stacks & Queues
- Elementary Sorting Methods (Ex: Selection sort & Insertion sort)
- Advanced Sorting Methods (Ex: Mergesort & Quicksort)
- Priority Queues (and Heap sort)
- Binary Trees
- ...etc.
